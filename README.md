**Overview**
-----------

A simple and efficient Contact Management System built using Java.
This project allows users to add, view, search, and delete contacts through a command-line interface. 
It is designed to help users organize their personal contacts in a structured and scalable way.

**Features**
-----------

*   **Add Contact**: Users can add new contacts with name, phone number, and email.
*   **View Contacts**: Users can view all saved contacts.
*   **Search Contacts**: Users can search for contacts by name, phone number, or email.
*   **Delete Contact**: Users can delete existing contacts.

**Technical Requirements**
-------------------------

*   **Language**: Java 8 or later
*   **IDE**: Eclipse, IntelliJ IDEA, or Visual Studio Code
*   **JDK**: Java Development Kit 8 or later
